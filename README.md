# Title

基于虚拟现实技术的智能导航系统

## Background
当今，导航软件已经成为人们日常出行中必不可少的工具，然而在实践中我们发现，日常导航方式（手机导航、车载导航等）都存在着以下几个问题：  
1.语音导航指路不够清晰明确；  
2.显示屏和实际路况难以结合导致的潜在安全问题。  

对于以上几个问题，基于虚拟现实技术，我们设计了智能可视化交互导航系统，希望能解决当前导航方式的不足。
